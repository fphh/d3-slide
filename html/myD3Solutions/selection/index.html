Selektionen und Transitionen

  1. Definieren Sie sich ein Array aus Buchstaben als Daten.
  2. Erzeugen Sie zunächst das Ausgangsbild mit Hilfe der enter-Selektion wie gewohnt. Um gemeinsame verschieben der Elemente zu ermöglichen, werden Sie das Gruppenelement benötigen.
  3. Ersetzen Sie Ihr Daten-Array durch ein anderes Array. Dabei können Buchstaben verschwinden, hinzukommen oder Ihre Position ändern. Damit die Buchstaben weiterhin identifiziert werden können, sollten Sie eine key-Funktion im Aufruf von data verwenden. Diese Funktion müssen Sie natürlich auch beim ersten Aufruf schon mitübergeben.
  4. Schreiben Sie Funktionen enter, update, enterAndUpdate und exit, die die jeweiligen Selektionen behandeln. Verzichten Sie zunächst einmal auf Transitionen.
  5. Rufen sie Ihre Funkionen nach einander mit Hilfe von setTimeout auf.
  6. Beachten Sie, dass die Reihenfolge der Aufrufe maßgeblich ist.
  7. Fügen Sie jetzt Transitionen ein, um eine Animation zu erhalten
Als zusätzliche Übung können sie einen Button definieren, der es ermöglicht, das Array mit den Buchstaben dynamisch zu erzeugen. Im Anschluß soll jeweils die Animation abgespielt werden.
