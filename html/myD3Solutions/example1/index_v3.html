Aufgabe 3 - Zwei Kreise als Markierung

Fügen Sie eine weitere Gruppe hinzu. In diese Gruppe fügen Sie zwei Kreise ein, einen für den inneren Rand und einen für den äußeren Rand. Sie können geimeinsame style-Attribute beider Kreise auch am Gruppenelement setzen (opacity, fill (none), stroke-width).

Im Callback für mouseenter setzen Sie die Radien der beiden Kreise. Den Index für das Array, dem Sie die Radien entnehmen, erhalten Sie dabei als zweiten Parameter des Callbacks. Achtung! Für den inneren Kreis kann der Index über die Array-Grenze hinausverweisen. In diesem Fall setzen sie Rahmenfarbe und Radius auf die des äußeren Kreises.

Durch die mit stroke-width gesetzte Dicke kann es sein, dass Sie beim überfahren eines Kreises den Eindruck haben, der Kreis würde größer. Um dieses Flackern zu verhinden, setzen Sie bei den farbigen Kreisen einfach stroke auf die Kreisfarbe und stroke-width auf die selbe Dicke wie bei den markierenden Kreisen.

Im Callback für mouseenter setzen sie die Gruppe sichtbar, im Callback für mouseleave unsichtbar.
